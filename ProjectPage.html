Projects

1). Taco Bell Parser

The Taco Bell Geolocator is a project that uses a CSV file with all the Taco Bell locations in the Alabama and finds the two with the farthest distance apart. First it will try to parse the latitute, longtitude, and name from the file. Once successful it will then store them in an instance to an interface in the program known as ITrackable. It will then take the two farthest locations in Alabama and return the Taco Bell locations, cities, and the distance in miles between the two.
